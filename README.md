# L.P.G : Loading Python Game
***
## 처음 실행 파일 :  index.py
## 실행 방법
* index.py를 연다
* Avoid Snow를 클릭하고 게임을 즐긴다.
* Brick Breaker를 클릭하고 게임을 즐긴다.
* Record를 클릭해 자신의 최고점수와 기록을 본다.
